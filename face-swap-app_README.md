# AI Face Swap App

Ini adalah aplikasi Face Swap dan Image Generator siap pakai yang berjalan 100% lokal di komputer Anda.

## Cara Menjalankan

### 1. Jalankan Backend (AI Server Python)
Buka terminal/Command Prompt, masuk ke folder `backend` di dalam folder ini:
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. Jalankan Frontend (User Interface)
Buka terminal baru, masuk ke folder `frontend`:
```bash
cd frontend
npm install
npm run dev
```
Buka browser dan akses URL lokal yang muncul (biasanya `http://localhost:5173`). Antarmuka dengan 2 kolom unggah dan ruang prompt sudah siap digunakan!
